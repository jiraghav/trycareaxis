# PI360 Website Master Plan

## Goal
Build a premium, enterprise-grade PI360 marketing website for PI law firms and clinics.

## Brand Direction
- Modern SaaS
- Premium and trustworthy
- Clean, spacious, conversion-focused
- Strong calls to action
- Professional, not gimmicky

## Target Audience
- Personal injury law firms
- Clinics and affiliate providers
- Admin teams using PI workflows

## Core Selling Points
- End-to-end PI case management
- Lawyer Portal
- Affiliate Portal
- Admin workflow visibility
- Scheduling, referrals, records, and updates in one place
- Nationwide operational feel

## Pages to Build
1. Homepage
2. Features
3. Lawyer Portal
4. Affiliate Portal
5. About / Why PI360
6. Contact / Book Demo

## Build Order
1. Improve homepage copy and layout
2. Add Features page
3. Add Lawyer Portal page
4. Add Affiliate Portal page
5. Add Contact / Book Demo page
6. Improve navbar/footer
7. Improve mobile responsiveness
8. Polish styling across all pages

## Rules
- Keep the app in Next.js
- Do not break the existing project
- Keep styling premium and clean
- Reuse components where possible
- Keep navigation simple and clear
- After each major step, explain what changed
